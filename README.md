# Ably Chat Reference Application

## Features

- Ably Chat SDK
- Next.js 14 App Directory
- Prisma | SQLite
- Radix UI Primitives / Shadcn
- Tailwind CSS
- Icons from [Lucide](https://lucide.dev)

## Usage

### Development

```bash
npm run dev
```

### Building

```bash
npm run build
```
